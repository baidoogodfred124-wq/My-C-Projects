## Features
- High- performing implementation of [Core Features]
- Modern C++ standards
- Cross-platform support (windows, macOS, Linux)
- Zero external dependencies
  
## Prerequisites
Before building the project, ensure you have the following installed:
-**compiler**: TDM-GCC
-**Dev-c++**: Embarcadero Dev-C++ v6.3 or newer is also recommended

## Setting up and Running the project
follow these steps to open and run the project layout inside your IDE
###1. Clone or Download the project
Download the repository files to your local machine and extract them.
###2. Open the Project File
1. Launch **Dev-c++**.
2. Go to **File** -> **Open Project or File...**
3. Navigate to the project folder and select the '.dev' project layout file
4. Click **Open**, Your source files will populate in the left sidebar.
###3. Copile and Run
-Press **F9** to compile the project,.
-Press **F10** to run the compiled executable.
-Press **F11** to **Compile & Run** in a single click.
The compiled '.exe' file will be generated directly iside your project folder.
